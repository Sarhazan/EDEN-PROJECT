# Context7 MCP - מדריך שימוש

## מה זה Context7?
Context7 הוא MCP server שמביא דוקומנטציה עדכנית ורשמית של ספריות ישירות לפרומפט שלך.

## איך להשתמש?
פשוט הוסף את המילים "use context7" בסוף הבקשה שלך.

### דוגמאות לפרויקט Eden:

#### WhatsApp Web.js
```
איך אני מטפל ב-QR code timeout ב-whatsapp-web.js? use context7
```

#### Socket.IO
```
מה האירועים הזמינים ב-Socket.IO 4.8 לניהול reconnection? use context7
```

#### Express 5.2
```
איך להגדיר middleware ב-Express 5.2? use context7
```

#### Better SQLite3
```
מה ה-API של prepared statements ב-better-sqlite3? use context7
```

## מה זה נותן?
✅ דוקומנטציה **מדויקת** ו**עדכנית** מהמקור הרשמי
✅ דוגמאות קוד **עובדות** במקום הלוצינציות
✅ תמיכה בגרסאות ספציפיות (למשל Socket.IO 4.8.3)

## מתי להשתמש?
- כשאתה לא בטוח ב-API של ספרייה
- כשיש גרסה חדשה ואתה צריך את התכונות החדשות
- כשאתה רוצה דוגמאות קוד רשמיות

### Stars System - Starring

- [x] **STAR-FUNC-01**: כפתור כוכב מוצג בכל כרטיס משימה בעמוד "היום שלי"
- [x] **STAR-FUNC-02**: כוכב זהוב מלא מוצג עבור משימות מסומנות (is_starred = true)
- [x] **STAR-FUNC-03**: כוכב אפור ריק מוצג עבור משימות רגילות (is_starred = false)
- [x] **STAR-FUNC-04**: click על כוכב עושה toggle של סטטוס starred
- [x] **STAR-FUNC-05**: שינוי starred נשמר ב-database (עמודה is_starred, טבלת tasks)
- [x] **STAR-FUNC-06**: עדכון starred משודר בזמן אמת דרך Socket.IO (אירוע task:updated)
- [x] **STAR-FUNC-07**: כוכב מוצג גם במשימות שהושלמו (בהיסטוריה)

### Stars System - Filtering

- [x] **STAR-FILT-01**: כפתור כוכב מוצג בכותרת "משימות לשבוע הקרוב" (ליד כפתור "שלח הכל")
- [x] **STAR-FILT-02**: כוכב מוחשך (אפור) = מצב הכל, כוכב זהוב = מצב סינון
- [x] **STAR-FILT-03**: click על כפתור כוכב עושה toggle בין "הכל" ל"רק מסומנים"
- [x] **STAR-FILT-04**: כשמסנן לפי כוכב, מוצגות רק משימות עם is_starred = true
- [x] **STAR-FILT-05**: משימות שהושלמו (completed) לא מופיעות גם כשמסנן לפי כוכב
- [x] **STAR-FILT-06**: מצב הסינון (כל/כוכב) נשמר ב-localStorage
- [x] **STAR-FILT-07**: סינון מתבצע לוקאלית (client-side) ללא קריאת API

### Resizable Columns

- [x] **RESIZE-01**: slider (מחוון) מוצג בין שתי העמודות בעמוד "היום שלי"
- [x] **RESIZE-02**: גרירה אופקית של slider משנה רוחב העמודות באופן דינמי
- [x] **RESIZE-03**: min-width: 250px נאכף לכל עמודה (למנוע עמודה צרה מדי)
- [x] **RESIZE-04**: max-width: 70% נאכף לכל עמודה (למנוע עמודה רחבה מדי)
- [x] **RESIZE-05**: cursor: col-resize מוצג בזמן hover על slider
- [x] **RESIZE-06**: קו מפריד ויזואלי (border) מסמן את ה-slider
- [x] **RESIZE-07**: רוחב עמודות נשמר ב-localStorage (key: myDayColumnWidths)
- [x] **RESIZE-08**: רוחב עמודות משוחזר אוטומטית בטעינת הדף
- [x] **RESIZE-09**: debounce של 100ms לשמירה ב-localStorage (למנוע writes מרובים)
- [x] **RESIZE-10**: כפתור "איפוס" מחזיר לברירת מחדל (50-50)
- [x] **RESIZE-11**: resizable columns מוצג רק בדסקטופ (>= 1024px)

### WhatsApp Integration - Server Integration

- [x] **WA-INTG-01**: WhatsApp client (whatsapp-web.js) runs inside main server process, not separate gateway
- [x] **WA-INTG-02**: QR code displayed in Settings page UI (browser), not terminal
- [x] **WA-INTG-03**: Disconnect button in Settings page allows manager to manually disconnect WhatsApp
- [x] **WA-INTG-04**: Always-on server (Railway) maintains persistent WhatsApp connection
- [x] **WA-INTG-05**: WhatsApp session persists across server restarts via LocalAuth (.wwebjs_auth directory)

### Mobile Responsiveness - Navigation

- [x] **MOB-NAV-01**: hamburger menu (☰) מוצג בפינה הימנית העליונה במובייל (< 768px)
- [x] **MOB-NAV-02**: Sidebar הרגיל מוסתר אוטומטית במובייל
- [x] **MOB-NAV-03**: click על hamburger פותח drawer מימין לשמאל (RTL) עם backdrop
- [x] **MOB-NAV-04**: drawer כולל את כל פריטי הניווט מהסיידבר + לוגו + logout
- [x] **MOB-NAV-05**: drawer נסגר ב-click על backdrop, כפתור X, או בחירת פריט ניווט
- [x] **MOB-NAV-06**: אנימציית slide-in/slide-out חלקה (300ms)
- [x] **MOB-NAV-07**: swipe מימין לשמאל סוגר את ה-drawer

### Mobile Responsiveness - Grids & Layouts

- [x] **MOB-GRID-01**: רשימת משימות ב"היום שלי" מוצגת ב-`grid-cols-1` במובייל
- [x] **MOB-GRID-02**: עמודות "משימות קבועות" ו"משימות לשבוע" stack vertically במובייל
- [x] **MOB-GRID-03**: stats bar למעלה מוצג ב-`grid-cols-2` במובייל (במקום 4)

- [x] **MOB-GRID-06**: רשימות בעמודים אחרים (עובדים, מערכות) ב-`grid-cols-1` במובייל
- [x] **MOB-GRID-07**: Timeline chart בעמוד "היום שלי" מותאם לגלילה אופקית במובייל
